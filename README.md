# Nairobi Weather Analysis (Jan 2025)

## Project Overview
Analyzed 7 days of hourly weather data for Nairobi, Kenya. Performed data cleaning, feature engineering, anomaly detection, and visualization.

## Key Findings
- Temperature range: 13.4°C to 28.0°C
- Warmest hour: 12:00 PM (noon)
- Coolest hour: 5:00 AM
- One heat anomaly detected (28.0°C on Jan 7)
- 21 hours of light rain (0.1-0.4mm)

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Skills Demonstrated
- Data loading and inspection
- Time series visualization
- Anomaly detection (2-standard deviation method)
- Feature engineering (hour extraction, rolling averages)
